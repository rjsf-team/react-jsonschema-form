# react-jsonschema-form

![Build Status](https://github.com/rjsf-team/react-jsonschema-form/workflows/CI/badge.svg)

一个简单易用，能够将 [JSON schema](http://json-schema.org/) 编译成HTML表单的 [React](https://reactjs.org/) 组件。

我们在 GitHub Pages 上托管了一个 [在线训练场](https://rjsf-team.github.io/react-jsonschema-form/)。

<a target='_blank' href='https://rjsf-team.github.io/react-jsonschema-form/'>
    <img alt='Playground' src='https://i.imgur.com/M8ZCES5.gif'/>
</a>

## Philosophy

react-jsonschema-form 旨在自动将 [JSON schema](http://json-schema.org/) 编译成 [React](https://reactjs.org/)HTML表单。如果你想要任何不可见的数据转换为表单，你只需要给我们一个 [JSON schema](http://json-schema.org/)，然后我们将会给你一个表单。如果您对数据有先验的了解，并且想要一个用于为其生成表单的工具包，则可以在其他地方寻找。

react-jsonschema-form 还附带了 `uiSchema` 和其他表单属性等工具用于自定义表单的外观。

## Installation

首先从 npm 安装依赖项，以及一个验证器实现安装包（例如）：@rjsf/validator-ajv8：

```bash
$ npm install @rjsf/core @rjsf/utils @rjsf/validator-ajv8 --save
```

然后导入依赖项，如下所示：

```ts
import validator from '@rjsf/validator-ajv8';
import Form from '@rjsf/core';
```

我们的最新版本需要 React 16+。您也可以安装适用于 React 15+ 的react-jsonschema-form（1.x 版本）。

## Usage

```tsx
import { RJSFSchema } from '@rjsf/utils';
import validator from '@rjsf/validator-ajv8';

const schema: RJSFSchema = {
  title: 'Todo',
  type: 'object',
  required: ['title'],
  properties: {
    title: { type: 'string', title: 'Title', default: 'A new task' },
    done: { type: 'boolean', title: 'Done?', default: false },
  },
};

const log = (type) => console.log.bind(console, type);

render(
  <Form
    schema={schema}
    validator={validator}
    onChange={log('changed')}
    onSubmit={log('submitted')}
    onError={log('errors')}
  />,
  document.getElementById('app')
);
```
